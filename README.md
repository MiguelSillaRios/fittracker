# 🏋️ FitTracker Pro

**App de seguimiento de entrenamientos construida con Claude Code en una mañana.**

Sin backend. Sin cuenta. Sin suscripción. Funciona directamente en el navegador y guarda todo en tu dispositivo.

🔗 **[Abrir app →](https://miguelsillarios.github.io/fittracker/)**

---

## ✨ Funcionalidades

- 📋 **Registro de entrenamientos** — series, repeticiones y peso por ejercicio
- 💪 **Biblioteca de 93 ejercicios** organizados por grupo muscular
- 📈 **Evolución de fuerza** — gráficas de 1RM estimado por ejercicio
- 📊 **Volumen y peso corporal** — seguimiento de tu progreso físico
- 🧮 **Cálculo automático de FFMI** — índice de masa libre de grasa
- 📁 **Exportación a Excel (.xlsx)** — comparte tu historial con tu entrenador
- 📱 **Diseño mobile-first** — pensada para usarla en el gym desde el móvil

## 🛠️ Tecnologías

| [SheetJS (xlsx)](https://sheetjs.com/) | Exportación a Excel |

## 🚀 Uso

No requiere instalación. Abre la app directamente:

👉 **[https://miguelsillarios.github.io/fittracker/](https://miguelsillarios.github.io/fittracker/)**

O clona el repo y abre `index.html` en tu navegador:

```bash
git clone https://github.com/MiguelSillaRios/fittracker.git
cd fittracker
open index.html
```

> ⚠️ Los datos se guardan en el `localStorage` de tu navegador. No se sincronizan entre dispositivos ni se envían a ningún servidor.
